# C++ Learning Projects

This repository contains a collection of C++ programs created for learning purposes.

## Programs

- **kasir.cpp** - Cashier/Point of Sale system
- **ipk.cpp** - GPA (Grade Point Average) calculator for students
- **tiketmasukpantai.cpp** - Beach ticket booking system
- **mahasiswa.cpp** - Student management program
- **belajar4.cpp, belajar5.cpp, belajar7.cpp** - Various C++ learning exercises
- **uts.cpp** - Midterm exam practice
- **UAS.cpp** - Final exam practice

## How to Compile and Run

```bash
g++ <filename>.cpp -o <output_name>
./<output_name>
```

Example:
```bash
g++ kasir.cpp -o kasir
./kasir
```
